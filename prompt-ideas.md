# Prompt: генерация идей применения плагина для физического Stream Deck

---

Я разрабатываю персональный плагин для физического устройства Ajazz AKP153E — это Stream Deck-совместимая панель с **18 кнопками** (сетка 5×3 + правая колонка дисплея). Каждая кнопка — 72×72 пикселя, между кнопками небольшие физические зазоры.

Плагин написан на **Rust** (openaction 2.6.0), запускается как демон на Linux/KDE Wayland. Профили описываются в TOML и генерируются в JSON для OpenDeck.

## Что уже реализовано

### Действия (actions)

1. **Stats** (`chs.deck.stats`)
   - Показывает CPU%, RAM%, количество Docker-контейнеров
   - Обновляется каждые 2–5 секунд
   - Выводит числовое значение поверх иконки

2. **Docker Count** (`chs.deck.docker.count`)
   - Количество запущенных Docker-контейнеров в реальном времени

3. **Media Playback** (`chs.deck.media`)
   - Управление через playerctl: play/pause, next, prev, stop
   - Показывает название трека и статус
   - Поддерживает любой MPRIS-плеер (Spotify, браузер, mpv и т.д.)

4. **Shell Command** (`chs.deck.shell`)
   - Запускает произвольную shell-команду при нажатии
   - Кастомный лейбл на кнопке

5. **Video Tile** (`chs.deck.video`)
   - Воспроизводит видеофайл через ffmpeg
   - Разбивает кадр на тайлы по нескольким кнопкам (например 5×3 = все 15 кнопок)
   - Поддержка fps, gap между кнопками, произвольного размера сетки
   - **Поддержка URL**: если путь — http(s) ссылка, использует yt-dlp для получения прямого потока

6. **Web Tile** (`chs.deck.webtile`)
   - Делает скриншот любого сайта через Playwright (headless Chromium)
   - Разбивает изображение на тайлы и отображает на кнопках
   - Настраиваемый интервал обновления (например каждые 30–60 секунд)
   - Учитывает физические зазоры между кнопками (gap_x, gap_y)
   - Работает с любым сайтом: погода, новости, дашборды, карты и т.д.

### Система профилей
- Несколько профилей, переключаются кнопками влево/вправо (кольцо)
- Слои (layers): общие элементы выносятся в отдельные файлы и накладываются
- Пример профилей: `work`, `weather`, `cinematic`
- Правая колонка: CPU%, RAM%, количество Docker-контейнеров + кнопка переключения профиля

### Upcoming: AI-controlled dashboard
Локальный HTTP-сервер отдаёт страницу с CSS grid 5×3 (72×72px ячейки). Claude через tool use может обновлять каждую ячейку независимо. Web Tile скриншотит localhost и отображает на деке. Shell action при нажатии кнопки отправляет событие на сервер → Claude реагирует.

## Контекст использования

- **Пользователь**: разработчик, работает в KDE на Linux
- **Устройство**: стоит рядом с монитором, всегда видно
- **Ограничения**: 72×72px на кнопку, 15 кнопок для контента + 3 для служебного (stats, переключение)
- **Сильные стороны**: любой сайт как тайл, видео/поток в реальном времени, shell-команды, AI управление
